
# 32-Bit ALU using Verilog

## Overview
This project implements a 32-Bit Arithmetic Logic Unit (ALU) using Verilog HDL.  
The ALU performs multiple arithmetic and logical operations based on the control signal provided.

The project is designed for FPGA implementation and simulation using Xilinx Vivado.

---

## Features
- 32-bit data processing
- Arithmetic operations
  - Addition
  - Subtraction
- Logical operations
  - AND
  - OR
  - XOR
  - NOT
- Shift operations
  - Left Shift
  - Right Shift
- Comparison operation
  - Set Less Than (SLT)
- Zero flag generation
